# Design Context

## Users
Recruiters evaluating candidates for full-time positions. They need to assess skills, experience, and design sensibility quickly to decide whether to move forward with an interview.

## Brand Personality
**Technical artistic creative** - blending precision with artistry. The portfolio demonstrates both technical mastery and creative vision.

## Design Principles
- maximum impact with minimum elements
- every element must earn its place
- editorial precision in spacing and typography
- unexpected compositions that break the grid

## Core Design Direction
- **Emotional goals**: competence, confidence, sophistication
- **Theme**: dark mode with lime green (#c5e431) accents
- **Aesthetic**: bold elegant minimal - refined, confident, uncluttered
- **Avoid**: AI slop, generic templates, predictable card grids, excessive decoration
- **Typography**: Inter (sans), Playfair Display (serif display) - current stack; use with precision and hierarchy
- **Motion**: purposeful, restrained; reveals hierarchy and state changes

This portfolio must feel intentionally designed—not generated. Every decision should answer: "why this element?"
